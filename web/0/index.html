Assignment Description

For this assignment we created an Amazon Web Services account, and set up a server instance to be used for the duration of this class. After we launched the server instance we ran several commands to set up programs such as mySQL and PHP. We also configured our individual development environments to be able to connect via SSH and FTP to our AWS instance Finally we created a sample webpage to demonstrate the success of our setup.
